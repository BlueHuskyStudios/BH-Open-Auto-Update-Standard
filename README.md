The Blue Automation for Renewing Code (The BARC)
================================================
A standard for implementing a system in which local applications may automatically poll a remote server for the purpose of updating said application.

Stage:
------
- Brainstorm
- **Refine**
- Mockup
- Lambda
- Alpha
- Beta
- Stable
